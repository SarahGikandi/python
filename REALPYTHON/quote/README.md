# 🌟 Random Motivation Quote Generator

A simple Python project that displays a random motivational quote every time the program runs.
This project was also connected to **Windows Task Scheduler** so a new quote appears automatically whenever the computer powers on.

---

## 📌 Features

* Displays a random motivational quote
* Uses Python’s built-in `random` module
* Beginner-friendly project
* Optional fullscreen GUI version using `tkinter`
* Can be automated to run at startup using Task Scheduler

---

## 🛠 Technologies Used

* Python
* `random`
* `tkinter` *(optional GUI version)*

---

## ▶️ How It Works

The program stores several motivational quotes inside a Python list.
Each time the script runs, one quote is randomly selected and displayed.

Example output:

```bash
Hi Jedz😎, Here is Your Motivation quote for Today🥰:

DREAM BIG AND DARE TO FAIL.
```

## 💻 Startup Automation

This project was connected to **Windows Task Scheduler** so it automatically runs when the computer starts.

That means every time the PC powers on, a random motivational quote is displayed.


## 🎨 GUI Version

The project also includes a commented-out `tkinter` version that can display quotes in fullscreen mode with a cleaner interface.

Features in the GUI version:

* Fullscreen window
* Centered quote display
* Press `ESC` to close
