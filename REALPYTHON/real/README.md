# Mood generator 🥰

A simple Python program that saves your daily moods into a text file with the current date and time.

## Features

* Records my mood
* Adds a timestamp automatically
* Saves entries to `mood.txt`
* Beginner-friendly Python project

## Example

```bash
How are you feeling today?
Happy and motivated

Mood saved! Take care 🥰
```

## Technologies Used

* Python
* File Handling
* Datetime Module
